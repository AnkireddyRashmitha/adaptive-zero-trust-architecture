Adaptive Zero Trust Architecture

This project implements an Adaptive Zero Trust Security model using Flask.

🔐 Features

- Continuous authentication
- Access control system
- Secure login interface
- Risk-based decisions

🛠️ Tech Stack

- Python (Flask)
- HTML/CSS
- SQLite

▶️ How to Run

pip install -r requirements.txt
python app.py

Then open:
http://127.0.0.1:5000

app.py  
templates/index.html
